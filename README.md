## 工厂自用管理平台_后端
> 暂定名称 "Shake", 实用为主, 高度个性化

### 功能模块

#### 基础管理
1. 往来企业, 企业联系人
2. 仓库, 库位
3. 物资信息
4. 数据字典

#### 业务管理
1. 收货登记
2. 发货登记
3. 库存分析

#### 财务管理
1. 应收账款
2. 应付账款
3. 结算登记

#### 系统管理
1. 菜单设置
2. 用户, 权限
3. 附件
4. 单据代码规则
